# Raindrops Barber Shop

Speculative redesign by NeonFrame Web Design for Raindrops Barber Shop, Las Vegas. 1829 E Charleston Blvd, 5.0 rating on 268 Google reviews, walk-ins welcome.

## Stack

- Pure HTML, CSS, vanilla JS
- Custom canvas rain animation (not a library, native `<canvas>`)
- GSAP + ScrollTrigger for scroll choreography
- Google Fonts: Anton, Inter, JetBrains Mono, Caveat via non-blocking load

## Pages

- `index.html` Hero with rain, services highlight, barbers, book, visit
- `services.html` Full cut list
- `404.html`

## What is cool about this one

- Live canvas rain over the hero with neon glow
- Neon "RAINDROPS" sign with flicker
- Custom cursor (razor blade glyph) on desktop
- Barber pole SVG with infinite stripe animation
- Click ripple where you click
- Scroll-triggered marquee that flips direction
